# Εργασία Git – Collaborative Workflow : “The Python Cookbook”

## Οδηγίες για φοιτητές

Καλώς ήρθατε στο repository της ομάδας zeta.

## Repository Structure

Το project είναι ενδεικτικό και περιέχει:

- `recipes/` folder που περιέχει ένα αρχείο 'example.py'
- ένα `main.py`
- βασικό skeleton για Python files

Η εργασία αφορά Git workflows, όχι προγραμματισμό.

---

# Στόχος

Practice a realistic Git collaboration workflow:

- Clone from GitHub  
- Work in a personal branch  
- Make small, meaningful commits  
- Push to the remote repository  
- Create a Pull Request to master  
- Resolve merge conflicts  
- Synchronize and clean branches  

---

# Οδηγίες Ανά Φοιτητή

Κάθε φοιτητής θα:

1. Κάνει clone το repo της ομάδας του
2. Εργαστεί αποκλειστικά σε προσωπικό branch  
   `recipe-<yourname>`
3. Κάνει **δύο commits**:
   - Προσθήκη προσωπικού python αρχείου στο `recipes/`
   - Προσθήκη ονόματος στη shared λίστα στο `main.py`
4. Κάνει push το branch στο remote
5. Ανοίξει Pull Request προς `master`
6. Επιλύσει merge conflicts αν χρειαστεί
7. Συγχωνεύσει το PR και καθαρίσει το branch

---
